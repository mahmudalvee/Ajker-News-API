While the var can have a global as well as functional scope (depending on the situation), the let and const have block scopes. Another difference is in updating the values and re-declaring the variables.

The variables declared with var can be updated and re-declaring while variables declared with let can be updated but cannot be re-declared. In the case of const, the variable cannot be updated or re-declared.

An arrow function is also known as the fat arrow function. It is a new feature introduced in ES6 that is a more concise syntax for writing function expressions. Both regular JavaScript functions and arrow functions work in a similar manner but there are some differences between them. Let's see the differences below:
  • Syntax
  • Arguments binding
  • 'this' keyword
  • Using a new keyword

  • map() creates a new array from calling a function for every array element. map() calls a function once for each element in an array.
  • forEach method is one of the several ways to loop through arrays. Each method has different features, and it is up to you, depending on what you're doing, to decide which one to use.
  • filter() method creates a new array filled with elements that pass a test provided by a function. It return an array of all values by filtering
  • find() method returns the value of the first element that passes a test. It executes a function for each array element. It returns undefined if no elements are found.
